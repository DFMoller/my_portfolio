Firebase Hosting

August 24, Monday

To put your web application online, it is necessary to find a host and there are many hosting services out there, but most are not free. Firebase hosting is a free hosting service, provided you stay inside the limits of the free account. These limits include a cap on storage, reads and writes, downloads etc. Firebase provides you with a default domain name for free and allows you to integrate your own domain name sourced elsewhere.

Firebase Functions

August 17, Monday

This week, it is worth looking at anther part of Google's Firebase. Firebase Functions are used to secure the most important parts of an application and in my first PWA, I used it to secure the functionality to give certain users admin privileges. Functions are not written in normal JavaScript files, but rather ones that are written to Google's specifications, secured through the command line upon deploying and, importantly, not linked to the rest of your files.
